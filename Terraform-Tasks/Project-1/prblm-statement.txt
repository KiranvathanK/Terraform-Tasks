Task 1 -

Create EC2 using terraform:
  1. Create a terraform template with EC2 resource (basic).
  2. Run terraform from your local machine.
  3. Modify the template for adding tag or any other values or properties and apply the changes.

Repeat the process for any other AWS resource like S3.
